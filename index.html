Resumen

Este proyecto tiene como objetivo mejorar la eficiencia del Sistema Eléctrico Nacional. Se ocuparan principalmente datos brindados por el CENACE, además de datos relevantes para determinar un modelo para el cálculo de energía, como los datos de la Comisión Nacional del Agua (CONAGUA) para el cambio de clima y estacionalidad en México, etc. Una vez extraidos, limpiados y almacendos, se mostrarán análisis de los datos obtenidos que ayudarán a tomar decisiones para el mejoramiento del sistema.

Introducción

El Centro Nacional de Control de Energía (CENACE) es un organismo público descentralizado cuyo objeto es ejercer el Control Operativo del Sistema Eléctrico Nacional. Fue creado para crear condiciones que promueven la competencia, eficiencia e imparcialidad, mediante la asignación y despacho óptimos de las Centrales Eléctricas para satisfacer la demanda de energía del Sistema Eléctrico Nacional.
